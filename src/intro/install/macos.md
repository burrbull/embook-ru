# macOS

Все инструменты можно установить с помощью [Homebrew]:

[Homebrew]: http://brew.sh/

``` console
$ # GDB
$ brew install armmbed/formulae/arm-none-eabi-gcc

$ # OpenOCD
$ brew install openocd

$ # QEMU
$ brew install qemu
```
